AthletePro Component Architecture

Overview

Critical Bug Fixes & Improvements

🐛 Fixed: Community Workout Routing Issue

Problem: Application crashed when users without community memberships accessed the Calendar page due to improper API route handling.

Solution: Implemented proper route precedence and community ID validation:

  • Added specific routes for /api/workouts/community and /api/workouts/community/
  • Moved community routes before generic /api/workouts/:id route
  • Added proper handling for undefined/null/invalid community IDs
  • Returns empty array instead of database errors for users without communities

Impact: All main application pages now load without errors.

API Endpoints

Authentication & Users

GET /api/auth/user - Get current authenticated user
POST /api/auth/register - Register new user with profile data
PATCH /api/user/profile - Update user profile information
GET /api/login - Local development authentication
GET /api/logout - User logout with session cleanup

Communities & Memberships

POST /api/communities - Create new community
GET /api/communities/my - Get user's managed community
GET /api/communities/:id - Get community details
GET /api/communities/:id/members - Get community members list
POST /api/communities/:id/members - Add member to community
DELETE /api/communities/:id/members/:userId - Remove community member
POST /api/communities/:id/announcements - Create community announcement
GET /api/communities/:id/announcements - Get community announcements
POST /api/communities/:id/attendance - Mark attendance for community
GET /api/communities/:id/attendance - Get attendance records
POST /api/communities/:id/goals - Create community goal
GET /api/communities/:id/goals - Get community goals
PATCH /api/goals/:id - Update goal achievement status

Workouts & Parsing

POST /api/workouts/parse - Parse workout text into structured data
POST /api/workouts - Create single workout
POST /api/workouts/bulk - Create multiple workouts from parsed data
GET /api/workouts/my - Get user's personal workouts
GET /api/workouts/community - Get community workouts (no community)
GET /api/workouts/community/ - Get community workouts (trailing slash)
GET /api/workouts/community/:id - Get specific community workouts
GET /api/workouts/:id - Get individual workout details

Workout Logs & Performance

POST /api/workout-logs - Log workout completion with performance data
GET /api/workout-logs/my - Get user's workout logs
GET /api/workout-logs/community/:id - Get community workout logs

Progress Tracking & Analytics

GET /api/progress/insights - Get personalized progress insights
GET /api/olympic-lifts/my - Get user's Olympic lift records
GET /api/olympic-lifts/progress/:liftName - Get lift progression data
GET /api/leaderboard/community/:id - Get community leaderboard rankings

Benchmark & Reference Data

GET /api/benchmark-workouts - Get benchmark workouts (filterable by category)
